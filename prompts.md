Роль: Ты — эксперт по машинному обучению в кибербезопасности, специализирующийся на сетях SDN.

Задача: Напиши Python-скрипт для построения классификатора обнаружения атак на основе датасета InSDN.

Детали датасета:

Файлы: Есть три файла: Normal_data.csv, OVS.csv, metasploitable-2.csv. В задаче нужно объединить их в один датафрейм для обучения.
Целевая колонка: В каждом файле есть столбец с типом трафика (например, DoS, DDoS, Probe, Botnet, Benign). Название столбца может отличаться. Определи его автоматически и объедини классы Benign/Normal в одну категорию. В файле Normal_data.csv все записи — нормальные.
Особенность: Датасет сгенерирован CICFlowMeter и содержит 252 столбца. Многие из них могут содержать бесконечности (inf) или пропуски.
Требования к скрипту:

Загрузка и объединение: Загрузи три CSV, объедини их, добавь колонку-источник (опционально). Проверь, что во всех файлах одинаковый набор колонок.
Предобработка:

Удали колонки, которые полностью пустые или содержат только одно уникальное значение.
Замени inf и -inf на NaN, затем заполни все NaN нулями.
Если есть временные метки или колонки, содержащие IP-адреса (например, Src IP, Dst IP), удали их, так как они могут привести к переобучению.
Целевая переменная: Создай целевую переменную Label. Все, что не Benign/Normal, пометь как Attack для бинарной классификации. Дополнительно: реализуй возможность многоклассовой классификации для определения конкретного типа атаки (используй существующие названия из столбца).
Балансировка: Используй SMOTE (из imblearn) только на тренировочной выборке.
Модель: Обучи Random Forest и XGBoost. Сравни их производительность.
Оценка: Выведи classification_report (precision, recall, f1-score) для бинарного и многоклассового сценариев. Построй матрицу ошибок (confusion matrix) в виде тепловой карты, чтобы видеть, какие типы атак путаются между собой.
Визуализация: Отобрази топ-20 наиболее важных признаков (feature importance) для лучшей модели.
Напиши код с подробными комментариями. Учти, что датасет может быть большим (>300k строк), поэтому добавь аргумент nrows для тестового запуска на части данных.
